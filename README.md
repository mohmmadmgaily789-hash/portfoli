# Noor Islam — Flutter MVP (Prayer Times & Qibla)

مشروع تجريبي لعرض مواقيت الصلاة وحساب اتجاه القبلة.

تشغيل:
1. تأكد من تثبيت Flutter.
2. انسخ الملفات إلى مشروع Flutter جديد.
3. نفّذ `flutter pub get`.
4. امنح صلاحية الموقع عند التشغيل.
5. شغّل `flutter run`.

ملاحظات:
- هذا MVP يحتوي على حسابات مبسطة. لتحسين الدقة استخدم مكتبات معيارية (مثل Adhan أو مكتبات مثبتة) وأضف إعدادات طرق الحساب.
- لإضافة إشعارات محلية، قم بإضافة حزمة `flutter_local_notifications`.
- لإضافة تشغيل مصحف صوتي وتنزيلات، استخدم `just_audio` و`dio` مع إدارة ملفات محلية.
- لإضافة AR والبوصلـة الحقيقية استخدم مكتبات ARKit/ARCore أو flutter_ar_plugin.
